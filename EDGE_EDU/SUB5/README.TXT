5. 외부 라이브러리 설명
 - 위치 
    : .\LIB
 - 패키지
    : com.lgcns.computing
 - 파일명
    : <Computing 라이브러리>.jar (예> LIB115.jar, LIB305.jar) 
 - 클래스명
    : ComputingLib
 - 메서드
    : public String compute(String param)
    : 파라미터 
      1) param - Device로부터 전달된 값

 - 사용방법
    : JAVA REFLECTION 기술을 이용하여 .jar파일을 로딩 후 compute 메서드 호출
